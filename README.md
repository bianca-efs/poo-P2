Descrição do problema resolvido
O sistema foi desenvolvido para auxiliar no gerenciamento de uma papelaria, permitindo o cadastro, consulta, atualização e exclusão de informações importantes para o funcionamento do negócio. A aplicação centraliza o controle de clientes, fornecedores, funcionários e produtos, facilitando a organização dos dados e reduzindo a necessidade de registros manuais.
---
Lista de entidades implementadas
* Cliente
* Fornecedor
* Funcionário
* Produto
---
Instruções para execução
1. Instalar o Java JDK compatível com o projeto.
2. Instalar e configurar o MySQL Server.
3. Criar o banco de dados denominado **papelaria**.
4. Configurar usuário e senha do banco de dados conforme definidos nas classes DAO do projeto.
5. Executar os scripts de criação das tabelas.
6. Abrir o projeto em uma IDE compatível (Eclipse ou VS Code).
7. Executar o comando: gradle run
8. Utilizar o menu principal para acessar as funcionalidades do sistema.
